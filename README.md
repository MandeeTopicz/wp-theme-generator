# wp-theme-generator

A WordPress theme generator powered by AI.

## Setup

```bash
pnpm install
```

## Scripts

```bash
pnpm dev        # Start all packages in dev mode
pnpm build      # Build all packages
pnpm test       # Run all tests
pnpm lint       # Lint all packages
pnpm typecheck  # Type-check all packages
```

## Packages

- `@wp-theme-gen/client` - React + Vite frontend
- `@wp-theme-gen/server` - Express API server
- `@wp-theme-gen/shared` - Shared types and validators
